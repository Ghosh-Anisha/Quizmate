# Quizmate
Free Quiz app to be able to conduct and share quizzes with other online



## Setup
1. Clone the repository and navigate into it
    ```sh
    git clone https://github.com/Ghosh-Anisha/Quizmate
    cd Quizmate
    ```
2. Install all dependencies
    ```sh
    npm install
    ```
3. Run the app
 ```sh
    npm start
    ```
